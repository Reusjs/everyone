## Install

```bash
$ npm install -save everyone
```

## Usage

### Use

```js
import everyone from 'everyone'	// 引入组件
Vue.use(everyone)
```
And import style manually:

```js
import 'everyone/dist/everyone.css'	// 引入css文件
```
